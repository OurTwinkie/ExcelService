##  Запуск
Склонировать репозиторий:
```bash
    git clone https://github.com/OurTwinkie/ExcelService.git
    cd ExcelService
```
Собрать проект:
```
    mvn package
```

Запустить:
```
    java -jar target/excel-service-1.0.0.jar
```

Ссылка для открытия:
```
    http://localhost:8080/swagger-ui/index.html
```
